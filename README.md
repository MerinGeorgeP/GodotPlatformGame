# 🎮 Platform Game

A simple 2D platformer built using Godot 4 and GDScript.

## 📖 About

This project is a beginner-friendly platform game where the player navigates through obstacles, jumps across platforms, collects cherries, and reaches the trophy at the end of the level to win.

The game was developed to learn core game development concepts such as:

- Player movement
- Jump mechanics
- Collision detection
- Collectibles
- Scene management
- Level design
- GDScript programming

## ✨ Features

- Smooth left and right movement
- Jumping mechanics
- Collectible cherries
- Win condition through trophy interaction
- Tile-based level design
- Pixel-art assets

## 🎮 Controls

| Action | Key |
|----------|----------|
| Move Left | A / Left Arrow |
| Move Right | D / Right Arrow |
| Jump | Space |

## 🛠️ Built With
